# 📱 WA Message Saver

WhatsApp deleted messages automatically save karne wali Android app.
**Notification Listener** use karke kaam karti hai — bilkul wैसे جیسے Play Store wali apps کرتی ہیں۔

---

## ✅ Features

- 💾 WhatsApp messages automatically save hoti hain
- 👤 User-wise contact list
- 🚫 Deleted messages clearly marked hoti hain  
- 📱 Text, Image, Video, Audio, Sticker — sab capture hota hai
- 🔄 Phone reboot ke baad bhi kaam karta hai
- 🌙 WhatsApp Business bhi support karta hai

## 🚀 APK Kaise Banayein (GitHub Actions — FREE)

### Step 1: GitHub Account Banayein
1. https://github.com par jayein
2. "Sign Up" click karein (free hai)

### Step 2: Naya Repository Banayein
1. GitHub par login karein
2. Green **"New"** button click karein
3. Repository name: `wa-message-saver`
4. **Public** select karein (free mein Actions chalein)
5. **"Create repository"** click karein

### Step 3: Files Upload Karein
1. Repository page par **"uploading an existing file"** click karein
2. Is project ka poora folder drag & drop karein
   - Sabhi files select karein (Ctrl+A)
   - GitHub par drop karein
3. **"Commit changes"** click karein

### Step 4: APK Automatically Build Hogi! 🎉
1. Upload hone ke baad, **"Actions"** tab click karein
2. "Build APK" workflow automatically shuru hoga
3. 3-5 minute wait karein
4. Build complete hone par **"Artifacts"** section mein APK milegi
5. Download karein aur install karein!

## 📲 App Install Karne ke Baad

### Permission Dein:
1. App open karein
2. "Open Settings" button click karein
3. Settings mein **"WA Message Saver"** dhundein
4. Toggle **ON** karein
5. Wapas app par ayein

### Bas! Ab:
- WhatsApp ka koi bhi message aane par automatically save ho jayega
- Sender message delete kare — app mein saved rahega
- App open karein → Contact select karein → Sare messages dekhein

---

## ⚠️ Important Notes

- App sirf **NEW** messages save karti hai (install ke baad se)
- Messages **aapke phone par hi** store hoti hain (privacy safe)
- WhatsApp notifications **ON** honi chahiye
- Chat **mute** nahi honi chahiye

---

## 🔧 Troubleshooting

**Messages save nahi ho rahe?**
- Settings > Apps > Special App Access > Notification Access > WA Message Saver = ON

**Purane deleted messages?**
- Ye possible nahi — sirf install ke baad wale save honge
